#  MERN Product Management App

##  Overview

This is a MERN stack application where users can manage products with full CRUD functionality.

---

##  Features

* Add product
* View products
* Edit product
* Delete product
* Image preview upload
* Responsive UI

---

##  Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB

---

##  API Endpoints

* GET /api/products
* POST /api/products
* PUT /api/products/:id
* DELETE /api/products/:id

---

##  How to Run

### Backend:

```bash
cd server
npm install
npm start
```

### Frontend:

```bash
cd client
npm install
npm run dev
```
